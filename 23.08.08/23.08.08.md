## pattern matching
### brute force
```python
p = "than"  # pattern
t = "better the last smile than the first laughter"  # text
M = len(p)  # pattern length
N = len(t)  # text length

def brute_force(p, t):
    i = o  # index of t
    j = 0  # index of p
    while j < M and i < N:
        if t[i] != p[j]:
            i = i - j
            j = -1
        i = i + 1
        j = j + 1
    if j == M:
        return i - M  # success
    else:
        return -1  # failure
```

time complexity
- O(MN)

### KMP algorithm
- Knuth Morris Pratt algorithm
- whenever we detect a mismatch, we take advantage of some of the chracters in the text of the next window to avoid matching the characters that we know

time complexity
- o(M + N)
